# 更新日志

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-02

### Added
- 飞书消息发送功能 (send_message)
- 群聊信息查询 (get_chat_info)
- 群成员列表获取 (get_chat_members)
- Webhook 创建
- 消息解析 (parse_message)
- 关键词自动回复处理器 (KeywordHandler)
- AI对话处理器 (AIHandler)
- OpenClaw Skill 封装
- CLI 命令行工具
- 完整使用文档

### Features
- 支持发送文本、rich_text、图片消息
- 支持发送给个人和群聊
- 支持接收消息事件
- 自动 Token 刷新
